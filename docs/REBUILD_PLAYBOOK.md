## 3. Cleanup Rules

### Remove These File Types

**Always delete** (unless in specific allowed locations):

- `.ipynb` - Jupyter notebooks (v1.0 artifacts)
- `.gz` - Compressed archives
- `.env` - Environment files (use `.env.example` templates)
- `.pem`, `.key` - Security keys (regenerate, don't commit)
- `.ico`, `.png`, `.jpg` - Images (unless in `/docs/assets/`)
- `.html` - Static HTML (v1.0 dashboards)
- `.md` - Markdown files (except in `/docs/` or root `README.md`)
- `Dockerfile*`, `docker-compose*.yml` - Docker configs (v1.0 deployment)
- `Makefile` - Build files (v1.0 workflow)
- `nginx.conf` - Server configs (v1.0 deployment)
- `*.backup`, `*.disabled` - Backup files

**Exceptions:**
- `/config/*.yml` - Configuration files (keep)
- `/docs/*.md` - Documentation (keep)
- `README.md` - Root documentation (keep)

### Keep These File Types

- `.py` - Python source files
- `.yaml`, `.yml` - Configuration in `/config/`
- `.jsonl`, `.json` - Data and config files actively used
- `.txt` - Requirements, licenses, structured data

### Archive Strategy

Move unused v1.0 artifacts to `/archive/v1_legacy/`:
```powershell
New-Item -ItemType Directory -Path "archive\v1_legacy"
Move-Item <unused_file> archive\v1_legacy\
```

## 6. v1.0 Fallback Safety Rule

### When Files Are Missing

> **If a file, class, or function is missing:**
> 
> 1. Check the original **SpiralCortex (v1.0)** repository
> 2. Copy it from there **unchanged**
> 3. Place it in the corresponding v2.0 folder following conventions above
> 4. Rename following naming conventions (remove phase numbers, etc.)
> 5. Update imports to match v2.0 structure
> 6. Run `pytest` to validate

## Appendix B: Verification Scripts

Create `tools/verify_structure.py`:

```python
#!/usr/bin/env python3
"""Verify SpiralCortex v2.0 structure matches canonical blueprint."""

import os
from pathlib import Path

REQUIRED_STRUCTURE = {
    'cortex': ['core', 'ethics', 'identity', 'temporal'],
    'codex': ['core', 'integration', 'learning', 'services', 'tax_engine', 'validation'],
    'nexus': ['core', 'learning', 'emotional', 'integration', 'validation'],
    'sensus': ['core', 'perception', 'integration', 'validation', 'services']
}

def verify_structure():
    """Check all required directories exist."""
    errors = []
    for lobe, subdirs in REQUIRED_STRUCTURE.items():
        for subdir in subdirs:
            path = Path(lobe) / subdir
            if not path.exists():
                errors.append(f"Missing: {path}")
    
    if errors:
        print("âŒ Structure verification failed:")
        for error in errors:
            print(f"  - {error}")
        return False
    else:
        print("âœ… Structure verification passed")
        return True

if __name__ == '__main__':
    import sys
    sys.exit(0 if verify_structure() else 1)
```
